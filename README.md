# MuchIQ

Muchiq es una aplicación de turismo inteligente que conecta a las personas con la riqueza cultural, gastronómica y urbana del norte del Perú (inicio en Lambayeque). Integra tecnología, cultura y comunidad para redescubrir el territorio desde lo local hasta lo turístico.

## Características
- Rutas personalizadas y recomendaciones inteligentes.
- Sistema de recompensas para fomentar la visita a sitios locales.
- Exploración de huacas, museos, festivales, restaurantes y rincones urbanos.
- Perfil de usuario y historial de visitas.
- Contenido cultural curado y colaborativo.

## Diseño y símbolo
El diseño está inspirado en el dragón mochica, símbolo de sabiduría y conexión entre mundos. Representa la unión entre el pasado ancestral y la innovación del presente.

## Cómo usar (instalación local rápida)
1. Coloca el proyecto en tu servidor local (ej. `C:/xampp/htdocs/MuchIQ`).
2. Ejecuta `npm i` y luego `npm run dev`
3. Abre en el navegador: `http://localhost/MuchIQ`

## Estructura sugerida (resumen)
- `/public` — archivos públicos (index.html, assets)
- `/src` — código fuente (frontend)
